Pertemuan 5

A. Materi : Variabel dan Operator Java
   1) Variabel = Tempat menampung nilai dalam memori
   2) Jenis Variabel
      a. Lokal = Digunakan dalam satu sub-segmen (method), tidak dapat menggunakan modifier
      b. Instance = Digunakan dalam kelas dan di luar sub-segmen, dapat menggunakan modifier
         - Metode Static
         - Metode Non-Static
      c. Static = Variabel kelas, tidak bisa diubah, keyword static, biasa digunakan untuk kelas lain
   3) Konstanta = Tidak bisa diubah, keyword final
   4) Operator
      a. Aritmatika = +, -, *, /, %, ++, --
      b. Relasional = >, <, ==, !=, >=, <=
      c. Logika = &&, ||, !
      d. Assignment = +=, -=, =, *=, /=, %=
      e. Miscellaneous
        - Kondisional = ((x == 1)? 11:10)
        - Instance Of = boolean var = obj1 instanceof obj2

A. Materi :Inheritance
   - Pengenalan Konsep Inheritance									
      Inheritance memungkinkan kita membuat kelas yang diturunkan dari kelas lain. Inheritance juga memungkinkan untuk menggunakan ulang kode program dan mempercepat proses implementasi, kelas yang menurunkan atribut dan fungsi disebut sebagai base class, parent, ascendant, kelas bapak, superclass.  
   - Pengenalan Keyword Super						
     berfungsi sebagai variable referensi class , yang digunakan untuk rujukan dari Super Class atau Parent class.
